Git and git hub

Git is an open-source, distributed version control system (VCS) that runs locally on your computer, while GitHub is a cloud-based hosting platform for Git repositories that provides collaboration and project management tools

Git: The Version Control Tool

Git allows you to track changes.

Git is the underlying technology that tracks changes in files, allowing developers to save "snapshots" (called commits) of their project's history over time.

Key features of Git:

Local Operation:Most operations are performed on your local machine, allowing you to work offline.

Branching:Enables developers to create isolated environments (branches) to experiment with new features or bug fixes without affecting the main codebase.
